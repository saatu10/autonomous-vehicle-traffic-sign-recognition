Section 02
Model Architecture
Custom CNN with two convolutional blocks + dropout regularisation, matching the Kaggle notebook achieving 98%+ test accuracy on GTSRB. Input: 30×30 RGB. Output: 43-class softmax.
🔵

Conv Block 1

Two Conv2D(32, 5×5, ReLU) layers → MaxPool(2×2) → Dropout(0.25). Captures low-level edges and colour gradients.

🟣

Conv Block 2

Two Conv2D(64, 3×3, ReLU) layers → MaxPool(2×2) → Dropout(0.25). Extracts higher-level shape and sign-specific features.

🟢

Classifier Head

Flatten → Dense(256, ReLU) → Dropout(0.5) → Dense(43, Softmax). Adam + categorical cross-entropy. LR scheduler on plateau.

python · gtsrb_model.py 
model = keras.Sequential([
  # Block 1
  Conv2D(32, (5,5), activation='relu'),
  Conv2D(32, (5,5), activation='relu'),
  MaxPooling2D((2,2)),
  Dropout(0.25),

  # Block 2
  Conv2D(64, (3,3), activation='relu'),
  Conv2D(64, (3,3), activation='relu'),
  MaxPooling2D((2,2)),
  Dropout(0.25),

  # Classifier
  Flatten(),
  Dense(256, activation='relu'),
  Dropout(0.5),
  Dense(43, activation='softmax'),
])

Training Strategy

🔄 Data Augmentation
Rotation ±10°, zoom 15%, shift 10%, shear 15% — simulates real-world variation in sign appearance.
📉 LR on Plateau
ReduceLROnPlateau halves LR when validation accuracy stalls, reaching a minimum of 1e-6.
💾 Best-checkpoint
ModelCheckpoint saves only the best validation-accuracy epoch to checkpoints/gtsrb_model.h5.
